works

  • foodkoko website

    サイトにアクセス
    説明
    授業で制作したAndroidアプリ、foodkokoを紹介するWebサイト
    制作期間
    2016年1月下旬 2週間
    使用技術
    • HTML/CSS
    • JavaScript(jQuery)
    • Vim
    • node.js
    • Sass
    • Autoprefixer
    工夫した点

    伝えたい情報は何か、を考え情報量をおさえる。

    情報量が減ってもスカスカに見えないよう、余白とフォントサイズを調整。

    また、見ていて違和感を覚えないよう、整列すべきコンポーネントは画像サイズから余白まできちんと揃えた。

    レスポンシブWebデザインでスマートフォンにも対応。

  • POP作成マクロ

    GitHub
    説明
    POP作成を支援するExcelマクロ
    制作期間
    2015年12月(2週間)～現在は機能追加を行っている
    開発経緯
    アルバイト先のスーパーマーケットにて、POP作成業務の効率が悪かった点を改善するため。
    工夫した点

    何度も同じデータを入力しなくてよい機能を実装。

    上の行と同じデータを省略可能。(自動的に上の行のデータが引き継がれる。)

    入力ミスを防ぐため、Excelシートに入力規則を設定。

    作業ミス防止のため、消されては困るシートなどを非表示に設定。

  • コトゲキ

    GitHub
    説明
    Windowsで動作するタイピングゲーム。間違ったキーの情報を確認できる。
    制作期間
    2014年6月～9月(3か月)
    開発経緯

    学校際で展示するゲームを制作するのでプログラムを担当。

    最初はシューティングゲームを作成するつもりで、プログラミングを進めていたものの、あとからタイピングゲームに変更したい、という要望があり、機能を追加。

    工夫した点

    ローマ字入力の処理では、ほとんどの入力パターンに対応。

    例:「くっちゃね」に対し「kucchane」「kuttyane」「kultutyane」など、なんでも認識可能。

    その他、ソフトウェアの設計、全体的なプログラミングを担当。（弾道計算のプログラミングはクラスメイトが担当)
